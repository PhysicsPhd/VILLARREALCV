En este proyecto mi intención fue crear una página web para mi CV personal.
Traté de utilizar la mayor cantidad de recursos aprendidos en este curso
que creí compatibles con el proyecto.
Es por lo anterior que decidí no incluir videos.
El formulario dice action="" porque todavía no he estudiado backend.
Las funciones de JavaScript fetchimginfo.js y geotolocalstorage.js no tienen un propósito funiconal en este proyecto,
simplemente las utilicé para practicar el manejo de:
JSON, ciclos iterativos, fetch(), geolocalizción del usuario y guardado de información en el localStorage.
Traté de utilizar imágenes de licencia libre y atribuí su origen en el modal correspondiente.
